# Better Pen ✏️

**Screen annotation built for coaches and VOD review.** Draw, highlight,
drop images, and spotlight what matters — live, over any game, video, or app.

Built by [Jiu](https://liquipedia.net/overwatch/Jiu), former professional
Overwatch coach, out of frustration with existing tools.

> ⬇️ **[Download the latest release →](../../releases/latest)**

---

## Features

**Drawing**
- Pen with smooth strokes, highlighter, laser pointer with fading trail
- Shapes: line, arrow, rectangle, triangle, circle — with fill, and
  editable corners (drag a rectangle into a parallelogram)
- Text labels and numbered / lettered stamps (per-colour counters:
  red 1-2, green 1-2 …)
- Strokes can auto-fade — three modes, including ink that ages as you draw

**Spotlight** 🔦
- Dim the whole screen, then *paint* the areas that stay bright —
  sightlines, chokes, off-angles glow out of the dark
- Draw on top of the dim; adjustable dim strength

**Images**
- Drop PNGs and animated GIFs straight onto the screen — agent icons,
  map callouts, emotes, whatever you build a library of
- Favourites grid with search, auto-scanned favourites folder
- Hotkey-bound images: press a key, the image lands at your cursor
- Move, resize, rotate, delete — all mouse-driven

**Workflow**
- Whiteboard / blackboard / maths-paper grid modes
- Pages — flip between annotation "slides" mid-session
- Save & load whole scenes to a file
- Fully rebindable: keyboard shortcuts *and* mouse buttons
  (erase on MB2, previous colour on MB4…) — works with Stream Deck
- Movable, snapping, customisable toolbar; collapses to a screen-edge tab
- Dark & light themes, multi-monitor support

## Install

1. Download **BetterPen-Setup-x.x.x.exe** from
   [Releases](../../releases/latest)
2. Windows SmartScreen will warn because the app isn't code-signed yet —
   click **More info → Run anyway**. (Signing is planned; indie tool tax.)
3. Run it — a short tour introduces the basics. The app lives in your
   system tray.

A portable `BetterPen.zip` (no install, settings stored beside the exe)
is also attached to each release.

## FAQ

**Is this safe with anti-cheat (Vanguard etc.)?**
Better Pen is a normal desktop overlay — it never injects into or reads
any game. It's designed for VOD review (replays, YouTube, recordings).
Note that games in *exclusive fullscreen* draw over any overlay; use
borderless/windowed.

**Where's the source code?**
Better Pen is closed-source freeware during the beta. This repo hosts
releases and documentation.

**Something's broken.**
Open an [issue](../../issues) or DM me — beta feedback is the whole
point. If the app crashed, attach the `crash.log` found next to the exe
(portable) or in `%APPDATA%\Better Pen` (installed).
